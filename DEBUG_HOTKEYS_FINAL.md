# ⌨️ Горячие клавиши Debug Console - Финальная версия

## 🎯 Все сочетания клавиш

| Клавиша | Действие | Контекст |
|---------|----------|----------|
| **Ctrl+Shift+D** | Открыть/закрыть Debug Console | В любой момент |
| **Ctrl+Enter** | Отправить отчёт | Когда консоль открыта |
| **Esc** | Закрыть консоль | Когда консоль открыта |

**Mac**: Используй `Cmd` вместо `Ctrl`

---

## 🚀 Быстрый рабочий процесс

### Полный цикл с клавиатуры:
```
1. Видишь проблему → Ctrl+Shift+D (открыл консоль)
2. Пишешь комментарий (описываешь проблему)
3. Ctrl+Enter (отправил отчёт)
4. ✅ Появится "Отчёт отправлен!"
5. Esc (закрыл консоль)
6. Продолжаешь работу
```

**Время:** ~10 секунд! ⚡

---

## 💡 Примеры использования

### Пример 1: Баллы не начислились
```
Действия:
1. Оплатил заказ
2. Вижу баллы = 0
3. Ctrl+Shift+D
4. Пишу: "Оплатил 1500₽, ожидал 150 баллов, показало 0"
5. Ctrl+Enter
6. Esc

Всё! Я получу отчёт с логами + твой комментарий
```

### Пример 2: Во время оформления заказа
```
Ситуация:
- Открыта модалка оплаты
- Вижу странную сумму

Действия:
1. Не закрывая модалку → Ctrl+Shift+D
2. Консоль откроется ПОВЕРХ модалки
3. Пишу: "Модалка показывает 1300₽, должно быть 1500₽"
4. Ctrl+Enter
5. Esc → консоль закрылась, модалка всё ещё открыта
6. Продолжаю оформление
```

### Пример 3: Быстрая отправка без комментария
```
Если хочешь просто отправить логи:
1. Ctrl+Shift+D
2. Ctrl+Enter (сразу, без комментария)
3. Esc

Готово! Логи отправлены
```

---

## 🛡️ Почему Ctrl+Enter, а не просто Enter?

**Проблема с Enter:**
- Enter нажимается случайно
- Enter используется для кнопок на сайте
- Enter может отправить форму на главной странице

**Преимущества Ctrl+Enter:**
- Стандарт в приложениях (Telegram, Slack, Gmail)
- Не конфликтует с сайтом
- Осознанное действие (две клавиши одновременно)

## 🎯 Итоговый рабочий процесс

### Когда возникает проблема с баллами:

```
1. Воспроизвёл проблему (оплатил, не начислились баллы)
   ↓
2. Ctrl+Shift+D (консоль открылась)
   ↓
3. Написал комментарий:
   "Оплатил заказ на 1500₽ картой.
    Ожидал: +150 баллов и уведомление
    Реально: баллы 0, уведомления нет"
   ↓
4. Ctrl+Enter (отправил)
   ↓
5. Esc (закрыл)
   ↓
6. Скопировал отчёт и показал мне:
   cat $(ls -t debug_reports/*_logs.txt | head -1)
   ↓
7. Я нашёл проблему и исправил! ✅
```

**Время на отправку отчёта: ~10 секунд**
