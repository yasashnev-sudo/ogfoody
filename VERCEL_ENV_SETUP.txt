3. После добавления ВСЕХ переменных - пересоберите проект (Redeploy)

ВАЖНО:
===========================================

3. Добавьте переменные для ВСЕХ окружений (Production, Preview, Development)
4. После добавления переменных ОБЯЗАТЕЛЬНО пересоберите проект
5. Проверьте результат на /api/diagnose

===========================================
КАК НАЙТИ ТОКЕН:
===========================================

1. Зайдите в ваш NocoDB
2. Settings → API Tokens
3. Создайте новый токен или скопируйте существующий

===========================================
КАК ПРОВЕРИТЬ ID ТАБЛИЦ:
===========================================

1. Откройте таблицу в NocoDB
2. Посмотрите в URL браузера - ID будет в формате md_xxxxx
3. Или используйте endpoint: https://ваш-домен.vercel.app/api/db/tables
