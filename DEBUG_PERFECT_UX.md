# ✅ ИДЕАЛЬНЫЙ UX: Автофокус + Горячие клавиши

## 🎯 Что добавлено

### **Автофокус на поле комментария** ⚡
При открытии Debug Console курсор **автоматически** в поле комментария!
- Не нужно кликать мышкой
- Сразу начинаешь печатать
- Максимально быстро!

---

## 🚀 Идеальный рабочий процесс

### Полностью с клавиатуры (0 кликов мышкой!):

```
1. Ctrl+Shift+D
   ↓ (консоль открылась, курсор УЖЕ в поле!)
   
2. Сразу пишешь: "Оплатил 1500₽, баллы 0"
   
3. Ctrl+Enter
   ↓ (отчёт отправлен ✅)
   
4. Esc
   ↓ (закрыл)

Готово! Время: ~5 секунд ⚡
```

**Ни одного клика мышкой!** 🎯

---

## ⌨️ Все горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **Ctrl+Shift+D** | Открыть консоль + курсор в поле |
| *(сразу печатаешь)* | Комментарий |
| **Ctrl+Enter** | Отправить отчёт |
| **Esc** | Закрыть консоль |

---

## 💡 Примеры использования

### Пример 1: Баллы не начислились (5 секунд!)
```
Действия:
1. Оплатил → вижу баллы = 0
2. Ctrl+Shift+D
3. (курсор УЖЕ в поле) → печатаю: "1500₽, баллы 0"
4. Ctrl+Enter
5. Esc

Готово! Отчёт отправлен с твоим комментарием!
```

### Пример 2: Уведомление не показалось
```
1. Ctrl+Shift+D
2. Печатаю: "Оплатил картой, уведомление не показалось"
3. Ctrl+Enter
4. Esc

Готово! 5 секунд!
```

### Пример 3: Быстро без комментария
```
Если хочешь просто отправить логи:
1. Ctrl+Shift+D
2. Ctrl+Enter (сразу, ничего не пишешь)
3. Esc

3 секунды! ⚡
```

## 💡 Почему это так удобно?

### 1. **Скорость** ⚡
- Не отвлекаешься на мышку
- Руки на клавиатуре
- Быстрая отправка отчёта

### 2. **Workflow** 🔄
```
Проблема → Ctrl+Shift+D → печатаешь → Ctrl+Enter → Esc → продолжаешь работу
```
Всё за 5 секунд!

### 3. **Контекст не теряется** 🎯
- Быстро описал проблему
- Логи свежие
- Ты помнишь, что произошло
